# Netflix Data Analysis

## Project Overview
This project analyzes a Netflix movie dataset to find insights about genres, popularity, and trends using Python.

## Questions Solved
1. Most frequent genre in dataset  
2. Genre with highest votes  
3. Most popular movie and its genre  
4. Least popular movie and its genre  
5. Year with highest number of movies  

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Drama is the most frequent genre  
- Drama has highest votes  
- "Spider-Man: No Way Home" is the most popular movie  
- Multi-genre movies are more popular  
- 2020 has highest number of releases  

## Files
- dataset.csv
